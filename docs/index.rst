Key-Value pairs scripts
=======================

The key-value pairs are annotations in OMERO useful to describe thoroughly the data and can be added & edited via the OMERO.web interface.

The scripts mentionned in this guide are here to provide additional support with key-value pairs, to import, export, delete or convert between namespaces by batch.
